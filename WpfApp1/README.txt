Перед первым запуском приложения требуется создать 3 переменных среды окружения:
1) ResultPath с путем к папке, куда попадают результаты резки
2) PDFPath с путем папки, где лежат макеты
3) HTMLPath с путем, где будут лежать обрезанные макеты для укладки на лист

Изменить пути можно будет в самой программе, важно, чтобы переменные просто были.
ВАЖНО: желательно(но не обязательно) перезапустить ПО после изменения путей к папкам.
